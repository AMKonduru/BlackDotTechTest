Having gone through the user story, the following assumptions have been made for a custom scenario for Web scraping:

The user is trying to search for a book online and wants to get an amalgamation of the search results.
The two search engines here have been assumed to be Wordery and BookDepository.

This has been designed as a basic WPF solution; wherein the user is expected to enter a search term and is inturn presented with a combined result set to compare the Prices.

Installed NuGet packages: 
Caliburn.Micro
HtmlAgilityPack
System.Net.Http
